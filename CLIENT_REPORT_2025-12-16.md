## üöÄ How to Use the News System

### Quick Start
1. The system runs automatically when the admin server starts
2. News is fetched every 24 hours
3. Articles appear on `/articles` page

### Manual Trigger
To fetch news immediately:
```
POST http://localhost:8080/api/news/fetch
```

### Check Status
```
GET http://localhost:8080/api/news/status
```

## ‚öôÔ∏è Configuration Required

### Environment Variables (apps/admin/.env)
```bash
# Enable automatic news fetching
NEWS_FETCH_ENABLED=true

# Articles to process per day (1-20)
NEWS_ARTICLES_PER_DAY=5

# Groq API Key (REQUIRED - get free at console.groq.com)
GROQ_API_KEY=your-groq-api-key

# NewsAPI Key (OPTIONAL - adds more sources)
NEWS_API_KEY=your-newsapi-key

# RSS Categories to fetch
RSS_CATEGORIES=mining,geology,metals,ree
```

### Environment Variables (apps/web/.env.local)
```bash
# API URL for fetching articles
NEXT_PUBLIC_API_URL=http://localhost:8080/api
```
